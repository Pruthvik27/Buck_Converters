# Open Loop Buck Converter (12V → 5V)

## Objective
Design a simple buck converter to step down 12V to 5V.

## Parameters
Vin = 12V  
Vout = 5V  
Load = 2.5Ω  
Switching Frequency = 50kHz  

## Duty Cycle
D = Vout / Vin = 0.416  
Used ≈ 0.45  

## Components
Inductor = 300µH  
Capacitor = 22µF  

## Observations
- Output voltage ≈ 5V  
- Ripple is minimal  
- System reaches steady state quickly  

## Learning
- Understood switching behavior  
- Learned duty cycle control  
- Observed inductor charging/discharging
